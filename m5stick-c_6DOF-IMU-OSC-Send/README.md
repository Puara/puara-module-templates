# M5Stick-C 6 DoF IMU OSC template 

This template demonstrates how to use Puara modules with the M5StickC to send the IMU data as OSC messages from the device.

Please refer to M5StickC documentation for further details on M5 functions. 
View [M5StickC Github repository](https://github.com/m5stack/M5StickC).

Please refer to CNMAT's OSC repository on Github for more details on OSC. 
View [CNMAT's OSC repo](https://github.com/CNMAT/OSC).
